# Práticas de Automação

Este repositório reúne pequenos projetos em Python voltados para automação de tarefas simples do dia a dia.  
O objetivo é praticar conceitos de programação e criar soluções úteis para organização e produtividade.

## Projetos incluídos
- [ProjetoFileOrganizer](./ProjetoFileOrganizer/README.md) → Organizador automático de arquivos por extensão.
- [ProjetoMergerPDF](./ProjetoMergerPDF/README.md) → Ferramenta para unir múltiplos PDFs em um único documento.

## Tecnologias utilizadas
- Python 3
- Bibliotecas padrão (os, shutil, tkinter)
- PyPDF2 (para manipulação de PDFs)
