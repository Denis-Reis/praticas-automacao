# Projeto MergerPDF

Este projeto em Python permite unir múltiplos arquivos PDF em um único documento final.  
É útil para consolidar relatórios, trabalhos acadêmicos ou qualquer conjunto de PDFs.

## Funcionalidades
- Lista todos os arquivos PDF em uma pasta.
- Ordena os arquivos por nome.
- Faz o merge em um único PDF final.

## Como usar
1. Coloque os arquivos PDF dentro da pasta `arquivos`.
2. Execute o script:
   ```bash
   python projeto1.py

## O resultado será um arquivo PDF Final.pdf com todos os documentos combinados.

##Tecnologias
- Python 3
- PyPDF2
